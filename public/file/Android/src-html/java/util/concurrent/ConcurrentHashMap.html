020    /**
021     * A hash table supporting full concurrency of retrievals and
022     * adjustable expected concurrency for updates. This class obeys the
023     * same functional specification as {@link java.util.Hashtable}, and
024     * includes versions of methods corresponding to each method of
025     * <tt>Hashtable</tt>. However, even though all operations are
026     * thread-safe, retrieval operations do <em>not</em> entail locking,
027     * and there is <em>not</em> any support for locking the entire table
028     * in a way that prevents all access.  This class is fully
029     * interoperable with <tt>Hashtable</tt> in programs that rely on its
030     * thread safety but not on its synchronization details.
031     *
032     * <p> Retrieval operations (including <tt>get</tt>) generally do not
033     * block, so may overlap with update operations (including
034     * <tt>put</tt> and <tt>remove</tt>). Retrievals reflect the results
035     * of the most recently <em>completed</em> update operations holding
036     * upon their onset.  For aggregate operations such as <tt>putAll</tt>
037     * and <tt>clear</tt>, concurrent retrievals may reflect insertion or
038     * removal of only some entries.  Similarly, Iterators and
039     * Enumerations return elements reflecting the state of the hash table
040     * at some point at or since the creation of the iterator/enumeration.
041     * They do <em>not</em> throw
042     * {@link ConcurrentModificationException}.  However, iterators are
043     * designed to be used by only one thread at a time.
044     *
045     * <p> The allowed concurrency among update operations is guided by
046     * the optional <tt>concurrencyLevel</tt> constructor argument
047     * (default 16), which is used as a hint for internal sizing.  The
048     * table is internally partitioned to try to permit the indicated
049     * number of concurrent updates without contention. Because placement
050     * in hash tables is essentially random, the actual concurrency will
051     * vary.  Ideally, you should choose a value to accommodate as many
052     * threads as will ever concurrently modify the table. Using a
053     * significantly higher value than you need can waste space and time,
054     * and a significantly lower value can lead to thread contention. But
055     * overestimates and underestimates within an order of magnitude do
056     * not usually have much noticeable impact. A value of one is
057     * appropriate when it is known that only one thread will modify
058     * and all others will only read.
059     *
060     * <p>This class implements all of the <em>optional</em> methods
061     * of the {@link Map} and {@link Iterator} interfaces.
062     *
063     * <p> Like {@link java.util.Hashtable} but unlike {@link
064     * java.util.HashMap}, this class does NOT allow <tt>null</tt> to be
065     * used as a key or value.
066     *
067     * @since 1.5
068     * @author Doug Lea
069     * @param <K> the type of keys maintained by this map
070     * @param <V> the type of mapped values 
071     */
072    public class ConcurrentHashMap<K, V> extends AbstractMap<K, V>
073            implements ConcurrentMap<K, V>, Serializable {
074        private static final long serialVersionUID = 7249069246763182397L;

138        /**
139         * Returns a hash code for non-null Object x.
140         * Uses the same hash code spreader as most other java.util hash tables.
141         * @param x the object serving as a key
142         * @return the hash code
143         */
144        static int hash(Object x) {
145            int h = x.hashCode();
146            h += ~(h << 9);
147            h ^=  (h >>> 14);
148            h +=  (h << 4);
149            h ^=  (h >>> 10);
150            return h;
151        }
152    
153        /**
154         * Returns the segment that should be used for key with given hash
155         * @param hash the hash code for the key
156         * @return the segment
157         */
158        final Segment<K,V> segmentFor(int hash) {
159            return (Segment<K,V>) segments[(hash >>> segmentShift) & segmentMask];
160        }

164        /**
165         * Segments are specialized versions of hash tables.  This
166         * subclasses from ReentrantLock opportunistically, just to
167         * simplify some locking and avoid separate construction.
168         **/
169        static final class Segment<K,V> extends ReentrantLock implements Serializable {
170            /*
171             * Segments maintain a table of entry lists that are ALWAYS
172             * kept in a consistent state, so can be read without locking.
173             * Next fields of nodes are immutable (final).  All list
174             * additions are performed at the front of each bin. This
175             * makes it easy to check changes, and also fast to traverse.
176             * When nodes would otherwise be changed, new nodes are
177             * created to replace them. This works well for hash tables
178             * since the bin lists tend to be short. (The average length
179             * is less than two for the default load factor threshold.)
180             *
181             * Read operations can thus proceed without locking, but rely
182             * on a memory barrier to ensure that completed write
183             * operations performed by other threads are
184             * noticed. Conveniently, the "count" field, tracking the
185             * number of elements, can also serve as the volatile variable
186             * providing proper read/write barriers. This is convenient
187             * because this field needs to be read in many read operations
188             * anyway. 
189             *
190             * Implementors note. The basic rules for all this are:
191             *
192             *   - All unsynchronized read operations must first read the
193             *     "count" field, and should not look at table entries if
194             *     it is 0.
195             *
196             *   - All synchronized write operations should write to
197             *     the "count" field after updating. The operations must not
198             *     take any action that could even momentarily cause
199             *     a concurrent read operation to see inconsistent
200             *     data. This is made easier by the nature of the read
201             *     operations in Map. For example, no operation
202             *     can reveal that the table has grown but the threshold
203             *     has not yet been updated, so there are no atomicity
204             *     requirements for this with respect to reads.
205             *
206             * As a guide, all critical volatile reads and writes are marked
207             * in code comments.
208             */
209

250             * Call only while holding lock or in constructor.

260            V get(Object key, int hash) {
261                if (count != 0) { // read-volatile
262                    HashEntry[] tab = table;
263                    int index = hash & (tab.length - 1);
264                    HashEntry<K,V> e = (HashEntry<K,V>) tab[index];
265                    while (e != null) {
266                        if (e.hash == hash && key.equals(e.key))
267                            return e.value;
268                        e = e.next;
269                    }
270                }
271                return null;
272            }
273    
274            boolean containsKey(Object key, int hash) {
275                if (count != 0) { // read-volatile
276                    HashEntry[] tab = table;
277                    int index = hash & (tab.length - 1);
278                    HashEntry<K,V> e = (HashEntry<K,V>) tab[index];
279                    while (e != null) {
280                        if (e.hash == hash && key.equals(e.key))
281                            return true;
282                        e = e.next;
283                    }
284                }
285                return false;
286            }
287    
288            boolean containsValue(Object value) {
289                if (count != 0) { // read-volatile
290                    HashEntry[] tab = table;
291                    int len = tab.length;
292                    for (int i = 0 ; i < len; i++)
293                        for (HashEntry<K,V> e = (HashEntry<K,V>)tab[i] ; e != null ; e = e.next)
294                            if (value.equals(e.value))
295                                return true;
296                }
297                return false;
298            }

300            boolean replace(K key, int hash, V oldValue, V newValue) {
301                lock();
302                try {
303                    int c = count;
304                    HashEntry[] tab = table;
305                    int index = hash & (tab.length - 1);
306                    HashEntry<K,V> first = (HashEntry<K,V>) tab[index];
307                    HashEntry<K,V> e = first;
308                    for (;;) {
309                        if (e == null)
310                            return false;
311                        if (e.hash == hash && key.equals(e.key))
312                            break;
313                        e = e.next;
314                    }
315    
316                    V v = e.value;
317                    if (v == null || !oldValue.equals(v))
318                        return false;
319    
320                    e.value = newValue;
321                    count = c; // write-volatile
322                    return true;
323                    
324                } finally {
325                    unlock();
326                }
327            }
328    
329            V replace(K key, int hash, V newValue) {
330                lock();
331                try {
332                    int c = count;
333                    HashEntry[] tab = table;
334                    int index = hash & (tab.length - 1);
335                    HashEntry<K,V> first = (HashEntry<K,V>) tab[index];
336                    HashEntry<K,V> e = first;
337                    for (;;) {
338                        if (e == null)
339                            return null;
340                        if (e.hash == hash && key.equals(e.key))
341                            break;
342                        e = e.next;
343                    }
344    
345                    V v = e.value;
346                    e.value = newValue;
347                    count = c; // write-volatile
348                    return v;
349                    
350                } finally {
351                    unlock();
352                }
353            }

356            V put(K key, int hash, V value, boolean onlyIfAbsent) {
357                lock();
358                try {
359                    int c = count;
360                    HashEntry[] tab = table;
361                    int index = hash & (tab.length - 1);
362                    HashEntry<K,V> first = (HashEntry<K,V>) tab[index];
363    
364                    for (HashEntry<K,V> e = first; e != null; e = (HashEntry<K,V>) e.next) {
365                        if (e.hash == hash && key.equals(e.key)) {
366                            V oldValue = e.value;
367                            if (!onlyIfAbsent)
368                                e.value = value;
369                            ++modCount;
370                            count = c; // write-volatile
371                            return oldValue;
372                        }
373                    }
374    
375                    tab[index] = new HashEntry<K,V>(hash, key, value, first);
376                    ++modCount;
377                    ++c;
378                    count = c; // write-volatile
379                    if (c > threshold)
380                        setTable(rehash(tab));
381                    return null;
382                } finally {
383                    unlock();
384                }
385            }
386    
387            HashEntry[] rehash(HashEntry[] oldTable) {
388                int oldCapacity = oldTable.length;
389                if (oldCapacity >= MAXIMUM_CAPACITY)
390                    return oldTable;
391    
392                /*
393                 * Reclassify nodes in each list to new Map.  Because we are
394                 * using power-of-two expansion, the elements from each bin
395                 * must either stay at same index, or move with a power of two
396                 * offset. We eliminate unnecessary node creation by catching
397                 * cases where old nodes can be reused because their next
398                 * fields won't change. Statistically, at the default
399                 * threshold, only about one-sixth of them need cloning when
400                 * a table doubles. The nodes they replace will be garbage
401                 * collectable as soon as they are no longer referenced by any
402                 * reader thread that may be in the midst of traversing table
403                 * right now.
404                 */
405    
406                HashEntry[] newTable = new HashEntry[oldCapacity << 1];
407                int sizeMask = newTable.length - 1;
408                for (int i = 0; i < oldCapacity ; i++) {
409                    // We need to guarantee that any existing reads of old Map can
410                    //  proceed. So we cannot yet null out each bin.
411                    HashEntry<K,V> e = (HashEntry<K,V>)oldTable[i];
412    
413                    if (e != null) {
414                        HashEntry<K,V> next = e.next;
415                        int idx = e.hash & sizeMask;
416    
417                        //  Single node on list
418                        if (next == null)
419                            newTable[idx] = e;
420    
421                        else {
422                            // Reuse trailing consecutive sequence at same slot
423                            HashEntry<K,V> lastRun = e;
424                            int lastIdx = idx;
425                            for (HashEntry<K,V> last = next;
426                                 last != null;
427                                 last = last.next) {
428                                int k = last.hash & sizeMask;
429                                if (k != lastIdx) {
430                                    lastIdx = k;
431                                    lastRun = last;
432                                }
433                            }
434                            newTable[lastIdx] = lastRun;
435    
436                            // Clone all remaining nodes
437                            for (HashEntry<K,V> p = e; p != lastRun; p = p.next) {
438                                int k = p.hash & sizeMask;
439                                newTable[k] = new HashEntry<K,V>(p.hash,
440                                                                 p.key,
441                                                                 p.value,
442                                                                 (HashEntry<K,V>) newTable[k]);
443                            }
444                        }
445                    }
446                }
447                return newTable;
448            }
449    
450            /**
451             * Remove; match on key only if value null, else match both.
452             */
453            V remove(Object key, int hash, Object value) {
454                lock();
455                try {
456                    int c = count;
457                    HashEntry[] tab = table;
458                    int index = hash & (tab.length - 1);
459                    HashEntry<K,V> first = (HashEntry<K,V>)tab[index];
460    
461                    HashEntry<K,V> e = first;
462                    for (;;) {
463                        if (e == null)
464                            return null;
465                        if (e.hash == hash && key.equals(e.key))
466                            break;
467                        e = e.next;
468                    }
469    
470                    V oldValue = e.value;
471                    if (value != null && !value.equals(oldValue))
472                        return null;
473    
474                    // All entries following removed node can stay in list, but
475                    // all preceding ones need to be cloned.
476                    HashEntry<K,V> newFirst = e.next;
477                    for (HashEntry<K,V> p = first; p != e; p = p.next)
478                        newFirst = new HashEntry<K,V>(p.hash, p.key,
479                                                      p.value, newFirst);
480                    tab[index] = newFirst;
481                    ++modCount;
482                    count = c-1; // write-volatile
483                    return oldValue;
484                } finally {
485                    unlock();
486                }
487            }
488    
489            void clear() {
490                lock();
491                try {
492                    HashEntry[] tab = table;
493                    for (int i = 0; i < tab.length ; i++)
494                        tab[i] = null;
495                    ++modCount;
496                    count = 0; // write-volatile
497                } finally {
498                    unlock();
499                }
500            }
501        }

524        /**
525         * Creates a new, empty map with the specified initial
526         * capacity and the specified load factor.
527         *
528         * @param initialCapacity the initial capacity. The implementation
529         * performs internal sizing to accommodate this many elements.
530         * @param loadFactor  the load factor threshold, used to control resizing.
531         * @param concurrencyLevel the estimated number of concurrently
532         * updating threads. The implementation performs internal sizing
533         * to try to accommodate this many threads.  
534         * @throws IllegalArgumentException if the initial capacity is
535         * negative or the load factor or concurrencyLevel are
536         * nonpositive.
537         */
538        public ConcurrentHashMap(int initialCapacity,
539                                 float loadFactor, int concurrencyLevel) {
540            if (!(loadFactor > 0) || initialCapacity < 0 || concurrencyLevel <= 0)
541                throw new IllegalArgumentException();
542    
543            if (concurrencyLevel > MAX_SEGMENTS)
544                concurrencyLevel = MAX_SEGMENTS;
545    
546            // Find power-of-two sizes best matching arguments
547            int sshift = 0;
548            int ssize = 1;
549            while (ssize < concurrencyLevel) {
550                ++sshift;
551                ssize <<= 1;
552            }
553            segmentShift = 32 - sshift;
554            segmentMask = ssize - 1;
555            this.segments = new Segment[ssize];
556    
557            if (initialCapacity > MAXIMUM_CAPACITY)
558                initialCapacity = MAXIMUM_CAPACITY;
559            int c = initialCapacity / ssize;
560            if (c * ssize < initialCapacity)
561                ++c;
562            int cap = 1;
563            while (cap < c)
564                cap <<= 1;
565    
566            for (int i = 0; i < this.segments.length; ++i)
567                this.segments[i] = new Segment<K,V>(cap, loadFactor);
568        }

592         * Creates a new map with the same mappings as the given map.  The
593         * map is created with a capacity of twice the number of mappings in
594         * the given map or 11 (whichever is greater), and a default load factor.
595         * @param t the map

604        // inherit Map javadoc
605        public boolean isEmpty() {
606            final Segment[] segments = this.segments;
607            /*
608             * We need to keep track of per-segment modCounts to avoid ABA
609             * problems in which an element in one segment was added and
610             * in another removed during traversal, in which case the
611             * table was never actually empty at any point. Note the
612             * similar use of modCounts in the size() and containsValue()
613             * methods, which are the only other methods also susceptible
614             * to ABA problems.
615             */
616            int[] mc = new int[segments.length];
617            int mcsum = 0;
618            for (int i = 0; i < segments.length; ++i) {
619                if (segments[i].count != 0)
620                    return false;
621                else 
622                    mcsum += mc[i] = segments[i].modCount;
623            }
624            // If mcsum happens to be zero, then we know we got a snapshot
625            // before any modifications at all were made.  This is
626            // probably common enough to bother tracking.
627            if (mcsum != 0) {
628                for (int i = 0; i < segments.length; ++i) {
629                    if (segments[i].count != 0 ||
630                        mc[i] != segments[i].modCount) 
631                        return false;
632                }
633            }
634            return true;
635        }
636    
637        // inherit Map javadoc
638        public int size() {
639            final Segment[] segments = this.segments;
640            int[] mc = new int[segments.length];
641            for (;;) {
642                long sum = 0;
643                int mcsum = 0;
644                for (int i = 0; i < segments.length; ++i) {
645                    sum += segments[i].count;
646                    mcsum += mc[i] = segments[i].modCount;
647                }
648                int check = 0;
649                if (mcsum != 0) {
650                    for (int i = 0; i < segments.length; ++i) {
651                        check += segments[i].count;
652                        if (mc[i] != segments[i].modCount) {
653                            check = -1; // force retry
654                            break;
655                        }
656                    }
657                }
658                if (check == sum) {
659                    if (sum > Integer.MAX_VALUE)
660                        return Integer.MAX_VALUE;
661                    else
662                        return (int)sum;
663                }
664            }
665        }

698        /**
699         * Returns <tt>true</tt> if this map maps one or more keys to the
700         * specified value. Note: This method requires a full internal
701         * traversal of the hash table, and so is much slower than
702         * method <tt>containsKey</tt>.
703         *
704         * @param value value whose presence in this map is to be tested.
705         * @return <tt>true</tt> if this map maps one or more keys to the
706         * specified value.
707         * @throws  NullPointerException  if the value is <tt>null</tt>.
708         */
709        public boolean containsValue(Object value) {
710            if (value == null)
711                throw new NullPointerException();
712    
713            final Segment[] segments = this.segments;
714            int[] mc = new int[segments.length];
715            for (;;) {
716                int sum = 0;
717                int mcsum = 0;
718                for (int i = 0; i < segments.length; ++i) {
719                    int c = segments[i].count;
720                    mcsum += mc[i] = segments[i].modCount;
721                    if (segments[i].containsValue(value))
722                        return true;
723                }
724                boolean cleanSweep = true;
725                if (mcsum != 0) {
726                    for (int i = 0; i < segments.length; ++i) {
727                        int c = segments[i].count;
728                        if (mc[i] != segments[i].modCount) {
729                            cleanSweep = false;
730                            break;
731                        }
732                    }
733                }
734                if (cleanSweep)
735                    return false;
736            }
737        }

1035        abstract class HashIterator {
1036            int nextSegmentIndex;
1037            int nextTableIndex;
1038            HashEntry[] currentTable;
1039            HashEntry<K, V> nextEntry;
1040            HashEntry<K, V> lastReturned;
1041    
1042            HashIterator() {
1043                nextSegmentIndex = segments.length - 1;
1044                nextTableIndex = -1;
1045                advance();
1046            }
1047    
1048            public boolean hasMoreElements() { return hasNext(); }
1049    
1050            final void advance() {
1051                if (nextEntry != null && (nextEntry = nextEntry.next) != null)
1052                    return;
1053    
1054                while (nextTableIndex >= 0) {
1055                    if ( (nextEntry = (HashEntry<K,V>)currentTable[nextTableIndex--]) != null)
1056                        return;
1057                }
1058    
1059                while (nextSegmentIndex >= 0) {
1060                    Segment<K,V> seg = (Segment<K,V>)segments[nextSegmentIndex--];
1061                    if (seg.count != 0) {
1062                        currentTable = seg.table;
1063                        for (int j = currentTable.length - 1; j >= 0; --j) {
1064                            if ( (nextEntry = (HashEntry<K,V>)currentTable[j]) != null) {
1065                                nextTableIndex = j - 1;
1066                                return;
1067                            }
1068                        }
1069                    }
1070                }
1071            }
1072    
1073            public boolean hasNext() { return nextEntry != null; }
1074    
1075            HashEntry<K,V> nextEntry() {
1076                if (nextEntry == null)
1077                    throw new NoSuchElementException();
1078                lastReturned = nextEntry;
1079                advance();
1080                return lastReturned;
1081            }
1082    
1083            public void remove() {
1084                if (lastReturned == null)
1085                    throw new IllegalStateException();
1086                ConcurrentHashMap.this.remove(lastReturned.key);
1087                lastReturned = null;
1088            }
1089        }

1296        /**
1297         * Save the state of the <tt>ConcurrentHashMap</tt>
1298         * instance to a stream (i.e.,
1299         * serialize it).
1300         * @param s the stream
1301         * @serialData
1302         * the key (Object) and value (Object)
1303         * for each key-value mapping, followed by a null pair.
1304         * The key-value mappings are emitted in no particular order.
1305         */
1306        private void writeObject(java.io.ObjectOutputStream s) throws IOException  {
1307            s.defaultWriteObject();
1308    
1309            for (int k = 0; k < segments.length; ++k) {
1310                Segment<K,V> seg = (Segment<K,V>)segments[k];
1311                seg.lock();
1312                try {
1313                    HashEntry[] tab = seg.table;
1314                    for (int i = 0; i < tab.length; ++i) {
1315                        for (HashEntry<K,V> e = (HashEntry<K,V>)tab[i]; e != null; e = e.next) {
1316                            s.writeObject(e.key);
1317                            s.writeObject(e.value);
1318                        }
1319                    }
1320                } finally {
1321                    seg.unlock();
1322                }
1323            }
1324            s.writeObject(null);
1325            s.writeObject(null);
1326        }

1328        /**
1329         * Reconstitute the <tt>ConcurrentHashMap</tt>
1330         * instance from a stream (i.e.,
1331         * deserialize it).
1332         * @param s the stream
1333         */
1334        private void readObject(java.io.ObjectInputStream s)
1335            throws IOException, ClassNotFoundException  {
1336            s.defaultReadObject();
1337    
1338            // Initialize each segment to be minimally sized, and let grow.
1339            for (int i = 0; i < segments.length; ++i) {
1340                segments[i].setTable(new HashEntry[1]);
1341            }
1342    
1343            // Read the keys and values, and put the mappings in the table
1344            for (;;) {
1345                K key = (K) s.readObject();
1346                V value = (V) s.readObject();
1347                if (key == null)
1348                    break;
1349                put(key, value);
1350            }
