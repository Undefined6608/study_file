066    /**
067     * The in-memory representation of a Java class. This representation serves as
068     * the starting point for querying class-related information, a process usually
069     * called "reflection". There are basically three types of {@code Class}
070     * instances: those representing real classes and interfaces, those representing
071     * primitive types, and those representing array classes.
072     * 
073     * <h4>Class instances representing object types (classes or interfaces)</h4>
074     * <p>
075     * These represent an ordinary class or interface as found in the class
076     * hierarchy. The name associated with these {@code Class} instances is simply
077     * the fully qualified class name of the class or interface that it represents.
078     * In addition to this human-readable name, each class is also associated by a
079     * so-called <em>signature</em>, which is the letter "L", followed by the
080     * class name and a semicolon (";"). The signature is what the runtime system
081     * uses internally for identifying the class (for example in a DEX file).
082     * </p>
083     * <h4>Classes representing primitive types</h4>
084     * <p>
085     * These represent the standard Java primitive types and hence share their
086     * names (for example "int" for the {@code int} primitive type). Although it is
087     * not possible to create new instances based on these {@code Class} instances,
088     * they are still useful for providing reflection information, and as the
089     * component type of array classes. There is one {@code Class} instance for each
090     * primitive type, and their signatures are:
091     * </p>
092     * <ul>
093     * <li>{@code B} representing the {@code byte} primitive type</li>
094     * <li>{@code S} representing the {@code short} primitive type</li>
095     * <li>{@code I} representing the {@code int} primitive type</li>
096     * <li>{@code J} representing the {@code long} primitive type</li>
097     * <li>{@code F} representing the {@code float} primitive type</li>
098     * <li>{@code D} representing the {@code double} primitive type</li>
099     * <li>{@code C} representing the {@code char} primitive type</li>
100     * <li>{@code Z} representing the {@code boolean} primitive type</li>
101     * <li>{@code V} representing void function return values</li>
102     * </ul>
103     * <p>
104     * <h4>Classes representing array classes</h4>
105     * <p>
106     * These represent the classes of Java arrays. There is one such {@code Class}
107     * instance per combination of array leaf component type and arity (number of
108     * dimensions). In this case, the name associated with the {@code Class}
109     * consists of one or more left square brackets (one per dimension in the array)
110     * followed by the signature of the class representing the leaf component type,
111     * which can be either an object type or a primitive type. The signature of a
112     * {@code Class} representing an array type is the same as its name. Examples
113     * of array class signatures are:
114     * </p>
115     * <ul>
116     * <li>{@code [I} representing the {@code int[]} type</li>
117     * <li>{@code [Ljava/lang/String;} representing the {@code String[]} type</li>
118     * <li>{@code [[[C} representing the {@code char[][][]} type (three dimensions!)</li>
119     * </ul>
120     * 
121     * @since Android 1.0
122     */

186        /**
187         * Returns a {@code Class} object which represents the class with the
188         * specified name. The name should be the name of a class as described in
189         * the {@link Class class definition}, however {@code Class}es representing
190         * primitive types can not be found using this method. Security rules will
191         * be obeyed.
192         * <p>
193         * If the class has not been loaded so far, it is being loaded and linked
194         * first. This is done through either the specified class loader or one of
195         * its parent class loaders. The caller can also request the class to be
196         * initialized, which means that a possible static initializer block is
197         * executed.  
198         * 
199         * @param className
200         *            the name of the non-primitive-type class to find.
201         * @param initializeBoolean
202         *            indicates whether the class should be initialized.
203         * @param classLoader
204         *            the class loader to use to load the class.
205         * @return the named {@code Class} instance.
206         * @throws ClassNotFoundException
207         *             if the requested class can not be found.
208         * @throws LinkageError
209         *             if an error occurs during linkage
210         * @throws ExceptionInInitializerError
211         *             if an exception occurs during static initialization of a
212         *             class.
213         * @since Android 1.0
214         */
215        public static Class<?> forName(String className, boolean initializeBoolean,
216                ClassLoader classLoader) throws ClassNotFoundException {
217            
218            if (classLoader == null) {
219                SecurityManager smgr = System.getSecurityManager();
220                if (smgr != null) {
221                    ClassLoader calling = VMStack.getCallingClassLoader();
222                    if (calling != null) {
223                        smgr.checkPermission(new RuntimePermission("getClassLoader"));
224                    }
225                }
226                
227                classLoader = ClassLoader.getSystemClassLoader();
228            }
229            // Catch an Exception thrown by the underlying native code. It wraps
230            // up everything inside a ClassNotFoundException, even if e.g. an
231            // Error occurred during initialization. This as a workaround for
232            // an ExceptionInInitilaizerError that's also wrapped. It is actually
233            // expected to be thrown. Maybe the same goes for other errors.
234            // Not wrapping up all the errors will break android though.
235            Class<?> result;
236            try {
237                result = classForName(className, initializeBoolean,
238                        classLoader);
239            } catch (ClassNotFoundException e) {
240                Throwable cause = e.getCause(); 
241                if (cause instanceof ExceptionInInitializerError) {
242                    throw (ExceptionInInitializerError) cause;
243                }
244                throw e;
245            }
246            return result;
247        }

306        /**
307         * Returns all the annotations of this class. If there are no annotations
308         * then an empty array is returned.
309         * 
310         * @return a copy of the array containing this class' annotations.
311         * @see #getDeclaredAnnotations()         
312         * @since Android 1.0
313         */
314        public Annotation[] getAnnotations() {
315            /*
316             * We need to get the annotations declared on this class, plus the
317             * annotations from superclasses that have the "@Inherited" annotation
318             * set.  We create a temporary map to use while we accumulate the
319             * annotations and convert it to an array at the end.
320             *
321             * It's possible to have duplicates when annotations are inherited.
322             * We use a Map to filter those out.
323             *
324             * HashMap might be overkill here.
325             */
326            HashMap<Class, Annotation> map = new HashMap<Class, Annotation>();
327            Annotation[] annos = getDeclaredAnnotations();
328    
329            for (int i = annos.length-1; i >= 0; --i)
330                map.put(annos[i].annotationType(), annos[i]);
331    
332            for (Class sup = getSuperclass(); sup != null;
333                    sup = sup.getSuperclass()) {
334                annos = sup.getDeclaredAnnotations();
335                for (int i = annos.length-1; i >= 0; --i) {
336                    Class clazz = annos[i].annotationType();
337                    if (!map.containsKey(clazz) &&
338                            clazz.isAnnotationPresent(Inherited.class)) {
339                        map.put(clazz, annos[i]);
340                    }
341                }
342            }
343    
344            /* convert annotation values from HashMap to array */
345            Collection<Annotation> coll = map.values();
346            return coll.toArray(new Annotation[coll.size()]);
347        }

349        /**
350         * Returns the canonical name of this class. If this class does not have a
351         * canonical name as defined in the Java Language Specification, then the
352         * method returns {@code null}.
353         * 
354         * @return this class' canonical name, or {@code null} if it does not have a
355         *         canonical name.
356         * @since Android 1.0
357         */
358        public String getCanonicalName() {
359            if (isLocalClass() || isAnonymousClass())
360                return null;
361    
362            if (isArray()) {
363                /*
364                 * The canonical name of an array type depends on the (existence of)
365                 * the component type's canonical name. 
366                 */
367                String name = getComponentType().getCanonicalName();
368                if (name != null) {
369                    return name + "[]";
370                } 
371            } else if (isMemberClass()) {
372                /*
373                 * The canonical name of an inner class depends on the (existence
374                 * of) the declaring class' canonical name. 
375                 */
376                String name = getDeclaringClass().getCanonicalName();
377                if (name != null) {
378                    return name + "." + getSimpleName();
379                }
380            } else {
381                /*
382                 * The canonical name of a top-level class or primitive type is
383                 * equal to the fully qualified name. 
384                 */
385                return getName();
386            }
387            
388            /*
389             * Other classes don't have a canonical name.
390             */
391            return null;
392        }

395         * Returns the class loader which was used to load the class represented by
396         * this {@code Class}. Implementations are free to return {@code null} for
397         * classes that were loaded by the bootstrap class loader. The Android
398         * reference implementation, though, returns a reference to an actual
399         * representation of the bootstrap class loader.
400         * 
401         * @return the class loader for the represented class.
402         * @throws SecurityException
403         *             if a security manager exists and it does not allow accessing
404         *             the class loader.
405         * @see ClassLoader
406         * @since Android 1.0
407         */
408        public ClassLoader getClassLoader() {
409            SecurityManager smgr = System.getSecurityManager();
410            ClassLoader loader = getClassLoaderImpl();
411            if (smgr != null && loader != null) {
412                ClassLoader calling = VMStack.getCallingClassLoader();
413    
414                if (calling != null && !calling.isAncestorOf(loader)) {
415                    smgr.checkPermission(new RuntimePermission("getClassLoader"));
416                }
417            }
418            
419            if (this.isPrimitive()) {
420                return null;
421            }
422            
423            if (loader == null) {
424                loader = BootClassLoader.getInstance();
425            }
426            
427            return loader;
428        }

546        /*
547         * Returns the list of member classes without performing any security checks
548         * first. This includes the member classes inherited from superclasses. If no
549         * member classes exist at all, an empty array is returned.
550         * 
551         * @param publicOnly reflects whether we want only public members or all of them
552         * @return the list of classes
553         */
554        private Class<?>[] getFullListOfClasses(boolean publicOnly) {
555            Class<?>[] result = getDeclaredClasses(this, publicOnly);
556            
557            // Traverse all superclasses
558            Class<?> clazz = this.getSuperclass();
559            while (clazz != null) {
560                Class<?>[] temp = getDeclaredClasses(clazz, publicOnly);
561                if (temp.length != 0) {
562                    result = arraycopy(new Class[result.length + temp.length], result, temp);
563                }
564                
565                clazz = clazz.getSuperclass();
566            }
567            
568            return result;
569        }

639        /*
640         * Finds a constructor with a given signature.
641         * 
642         * @param list the list of constructors to search through
643         * @param parameterTypes the formal parameter list
644         * @return the matching constructor
645         * @throws NoSuchMethodException if the constructor does not exist.
646         */
647        private Constructor<T> getMatchingConstructor(
648                Constructor<T>[] list, Class<?>[] parameterTypes)
649                throws NoSuchMethodException {
650            for (int i = 0; i < list.length; i++) {
651                if (compareClassLists(list[i].getParameterTypes(), parameterTypes)) {
652                    return list[i];
653                }
654            }
655            
656            // BEGIN android-changed
657            StringBuilder sb = new StringBuilder();
658            sb.append(getSimpleName());
659            sb.append('(');
660            boolean first = true;
661            for (Class<?> p : parameterTypes) {
662                if (!first) {
663                    sb.append(',');
664                }
665                first = false;
666                sb.append(p.getSimpleName());
667            }
668            sb.append(')');
669            throw new NoSuchMethodException(sb.toString());
670            // END android-changed
671        }

1590        /**
1591         * This must be provided by the vm vendor, as it is used by other provided
1592         * class implementations in this package. This method is used by
1593         * SecurityManager.classDepth(), and getClassContext() which use the
1594         * parameters (-1, false) and SecurityManager.classLoaderDepth(),
1595         * currentClassLoader(), and currentLoadedClass() which use the parameters
1596         * (-1, true). Walk the stack and answer an array containing the maxDepth
1597         * most recent classes on the stack of the calling thread. Starting with the
1598         * caller of the caller of getStackClasses(), return an array of not more
1599         * than maxDepth Classes representing the classes of running methods on the
1600         * stack (including native methods). Frames representing the VM
1601         * implementation of java.lang.reflect are not included in the list. If
1602         * stopAtPrivileged is true, the walk will terminate at any frame running
1603         * one of the following methods: <code><ul>
1604         * <li>java/security/AccessController.doPrivileged(Ljava/security/PrivilegedAction;)Ljava/lang/Object;</li>
1605         * <li>java/security/AccessController.doPrivileged(Ljava/security/PrivilegedExceptionAction;)Ljava/lang/Object;</li>
1606         * <li>java/security/AccessController.doPrivileged(Ljava/security/PrivilegedAction;Ljava/security/AccessControlContext;)Ljava/lang/Object;</li>
1607         * <li>java/security/AccessController.doPrivileged(Ljava/security/PrivilegedExceptionAction;Ljava/security/AccessControlContext;)Ljava/lang/Object;</li>
1608         * </ul></code> If one of the doPrivileged methods is found, the walk terminate
1609         * and that frame is NOT included in the returned array. Notes:
1610         * <ul>
1611         * <li>This method operates on the defining classes of methods on stack.
1612         * NOT the classes of receivers.</li>
1613         * <li>The item at index zero in the result array describes the caller of
1614         * the caller of this method.</li>
1615         * </ul>
1616         * 
1617         * @param maxDepth
1618         *            maximum depth to walk the stack, -1 for the entire stack
1619         * @param stopAtPrivileged

1622         */
