045    /**
046     * Produces concatenated
047     * messages in language-neutral way. Use this class to construct messages 
048     * displayed for end users.
049     * <p>
050     * {@code MessageFormat} takes a set of objects, formats them and then
051     * inserts the formatted strings into the pattern at the appropriate places.
052     * </p>
053     * <p>
054     * <strong>Note:</strong> {@code MessageFormat} differs from the other
055     * {@code Format} classes in that you create a {@code MessageFormat}
056     * object with one of its constructors (not with a {@code getInstance}
057     * style factory method). The factory methods aren't necessary because
058     * {@code MessageFormat} itself doesn't implement locale specific
059     * behavior. Any locale specific behavior is defined by the pattern that you
060     * provide as well as the subformats used for inserted arguments.
061     * 
062     * <h4><a name="patterns">Patterns and their interpretation</a></h4>
063     * 
064     * {@code MessageFormat} uses patterns of the following form:
065     * <blockquote>
066     * 
067     * <pre>
068     * <i>MessageFormatPattern:</i>
069     *         <i>String</i>
070     *         <i>MessageFormatPattern</i> <i>FormatElement</i> <i>String</i>
071     * <i>FormatElement:</i>
072     *         { <i>ArgumentIndex</i> }
073     *         { <i>ArgumentIndex</i> , <i>FormatType</i> }
074     *         { <i>ArgumentIndex</i> , <i>FormatType</i> , <i>FormatStyle</i> }
075     * <i>FormatType: one of </i>
076     *         number date time choice
077     * <i>FormatStyle:</i>
078     *         short
079     *         medium
080     *         long
081     *         full
082     *         integer
083     *         currency
084     *         percent
085     *         <i>SubformatPattern</i>
086     * <i>String:</i>
087     *         <i>StringPart&lt;sub&gt;opt&lt;/sub&gt;</i>
088     *         <i>String</i> <i>StringPart</i>
089     * <i>StringPart:</i>
090     *         ''
091     *         ' <i>QuotedString</i> '
092     *         <i>UnquotedString</i>
093     * <i>SubformatPattern:</i>
094     *         <i>SubformatPatternPart&lt;sub&gt;opt&lt;/sub&gt;</i>
095     *         <i>SubformatPattern</i> <i>SubformatPatternPart</i>
096     * <i>SubFormatPatternPart:</i>
097     *         ' <i>QuotedPattern</i> '
098     *         <i>UnquotedPattern</i>
099     * </pre>
100     * 
101     * </blockquote>
102     * 
103     * <p>
104     * Within a <i>String</i>, {@code "''"} represents a single quote. A
105     * <i>QuotedString</i> can contain arbitrary characters except single quotes;
106     * the surrounding single quotes are removed. An <i>UnquotedString</i> can
107     * contain arbitrary characters except single quotes and left curly brackets.
108     * Thus, a string that should result in the formatted message "'{0}'" can be
109     * written as {@code "'''{'0}''"} or {@code "'''{0}'''"}.
110     * <p>
111     * Within a <i>SubformatPattern</i>, different rules apply. A <i>QuotedPattern</i>
112     * can contain arbitrary characters except single quotes, but the surrounding
113     * single quotes are <strong>not</strong> removed, so they may be interpreted
114     * by the subformat. For example, {@code "{1,number,$'#',##}"} will
115     * produce a number format with the hash-sign quoted, with a result such as:
116     * "$#31,45". An <i>UnquotedPattern</i> can contain arbitrary characters except
117     * single quotes, but curly braces within it must be balanced. For example,
118     * {@code "ab {0} de"} and {@code "ab '}' de"} are valid subformat
119     * patterns, but {@code "ab {0'}' de"} and {@code "ab } de"} are
120     * not.
121     * </p>
122     * <dl>
123     * <dt><b>Warning:</b></dt>
124     * <dd>The rules for using quotes within message format patterns unfortunately
125     * have shown to be somewhat confusing. In particular, it isn't always obvious
126     * to localizers whether single quotes need to be doubled or not. Make sure to
127     * inform localizers about the rules, and tell them (for example, by using
128     * comments in resource bundle source files) which strings will be processed by
129     * {@code MessageFormat}. Note that localizers may need to use single quotes in
130     * translated strings where the original version doesn't have them. <br>
131     * Note also that the simplest way to avoid the problem is to use the real
132     * apostrophe (single quote) character \u2019 (') for human-readable text, and
133     * to use the ASCII apostrophe (\u0027 ' ) only in program syntax, like quoting
134     * in {@code MessageFormat}. See the annotations for U+0027 Apostrophe in The Unicode
135     * Standard.
136     * </dl>
137     * <p>
138     * The <i>ArgumentIndex</i> value is a non-negative integer written using the
139     * digits '0' through '9', and represents an index into the
140     * {@code arguments} array passed to the {@code format} methods or
141     * the result array returned by the {@code parse} methods.
142     * <p>
143     * The <i>FormatType</i> and <i>FormatStyle</i> values are used to create a
144     * {@code Format} instance for the format element. The following table
145     * shows how the values map to {@code Format} instances. Combinations not shown in the
146     * table are illegal. A <i>SubformatPattern</i> must be a valid pattern string
147     * for the {@code Format} subclass used.
148     * <p>
149     * <table border=1>
150     * <tr>
151     * <th>Format Type</th>
152     * <th>Format Style</th>
153     * <th>Subformat Created</th>
154     * </tr>
155     * <tr>
156     * <td colspan="2"><i>(none)</i></td>
157     * <td>{@code null}</td>
158     * </tr>
159     * <tr>
160     * <td rowspan="5">{@code number}</td>
161     * <td><i>(none)</i></td>
162     * <td>{@code NumberFormat.getInstance(getLocale())}</td>
163     * </tr>

175     * </tr>

195     * <td>{@code DateFormat.getDateInstance(DateFormat.LONG, getLocale())}</td>

207     * <td><i>(none)</i></td>
208     * <td>{@code DateFormat.getTimeInstance(DateFormat.DEFAULT, getLocale())}</td>
209     * </tr>

212     * <td>{@code DateFormat.getTimeInstance(DateFormat.SHORT, getLocale())}</td>

216     * <td>{@code DateFormat.getTimeInstance(DateFormat.DEFAULT, getLocale())}</td>
217     * </tr>
218     * <tr>
219     * <td>{@code long}</td>
220     * <td>{@code DateFormat.getTimeInstance(DateFormat.LONG, getLocale())}</td>
221     * </tr>
222     * <tr>
223     * <td>{@code full}</td>
224     * <td>{@code DateFormat.getTimeInstance(DateFormat.FULL, getLocale())}</td>
225     * </tr>
226     * <tr>
227     * <td><i>SubformatPattern</i></td>
228     * <td>{@code new SimpleDateFormat(subformatPattern, getLocale())}</td>
229     * </tr>
230     * <tr>
231     * <td>{@code choice}</td>
232     * <td><i>SubformatPattern</i></td>
233     * <td>{@code new ChoiceFormat(subformatPattern)}</td>
234     * </tr>
235     * </table>
236     * 
237     * <h4>Usage Information</h4>
238     * <p>
239     * Here are some examples of usage: <blockquote>
240     * 
241     * <pre>
242     * Object[] arguments = {
243     *         new Integer(7), new Date(System.currentTimeMillis()),
244     *         "a disturbance in the Force"};
245     * String result = MessageFormat.format(
246     *         "At {1,time} on {1,date}, there was {2} on planet {0,number,integer}.",
247     *         arguments);
248     * <em>
249     * Output:
250     * </em>
251     * At 12:30 PM on Jul 3, 2053, there was a disturbance in the Force on planet 7.
252     * </pre>
253     * 
254     * </blockquote> 
255     * <p>
256     * Typically, the message format will come from resources, and the
257     * arguments will be dynamically set at runtime.
258     * </p>
259     * <p>
260     * Example 2: <blockquote>
261     * 
262     * <pre>
263     * Object[] testArgs = {new Long(3), "MyDisk"};
264     * MessageFormat form = new MessageFormat("The disk \"{1}\" contains {0} file(s).");
265     * System.out.println(form.format(testArgs));
266     * <em>
267     * Output with different testArgs:
268     * </em>
269     * The disk "MyDisk" contains 0 file(s).
270     * The disk "MyDisk" contains 1 file(s).
271     * The disk "MyDisk" contains 1,273 file(s).
272     * </pre>
273     * 
274     * </blockquote>
275     * 
276     * <p>
277     * For more sophisticated patterns, you can use a {@code ChoiceFormat} to
278     * get output such as: 
279     * </p>
280     * <blockquote>
281     * 
282     * <pre>
283     * MessageFormat form = new MessageFormat("The disk \"{1}\" contains {0}.");
284     * double[] filelimits = {0,1,2};
285     * String[] filepart = {"no files","one file","{0,number} files"};
286     * ChoiceFormat fileform = new ChoiceFormat(filelimits, filepart);
287     * form.setFormatByArgumentIndex(0, fileform);
288     * Object[] testArgs = {new Long(12373), "MyDisk"};
289     * System.out.println(form.format(testArgs));
290     * <em>
291     * Output (with different testArgs):
292     * </em>
293     * The disk "MyDisk" contains no files.
294     * The disk "MyDisk" contains one file.
295     * The disk "MyDisk" contains 1,273 files.
296     * </pre>
297     * 
298     * </blockquote> You can either do this programmatically, as in the above
299     * example, or by using a pattern (see {@link ChoiceFormat} for more
300     * information) as in: <blockquote>
301     * 
302     * <pre>
303     * form.applyPattern("There {0,choice,0#are no files|1#is one file|1&lt;are {0,number,integer} files}.");
304     * </pre>
305     * 
306     * </blockquote>
307     * <p>
308     * <strong>Note:</strong> As we see above, the string produced by a
309     * {@code ChoiceFormat} in {@code MessageFormat} is treated
310     * specially; occurances of '{' are used to indicated subformats, and cause
311     * recursion. If you create both a {@code MessageFormat} and
312     * {@code ChoiceFormat} programmatically (instead of using the string
313     * patterns), then be careful not to produce a format that recurses on itself,
314     * which will cause an infinite loop.
315     * </p>
316     * <p>
317     * When a single argument is parsed more than once in the string, the last match
318     * will be the final result of the parsing. For example:
319     * </p>
320     * <blockquote>
321     * <pre>
322     * MessageFormat mf = new MessageFormat("{0,number,#.##}, {0,number,#.#}");
323     * Object[] objs = {new Double(3.1415)};
324     * String result = mf.format(objs);
325     * // result now equals "3.14, 3.1"
326     * objs = null;
327     * objs = mf.parse(result, new ParsePosition(0));
328     * // objs now equals {new Double(3.1)}
329     * </pre>
330     * </blockquote>
331     * <p>
332     * Likewise, parsing with a {@code MessageFormat} object using patterns 
333     * containing multiple occurrences of the same argument would return the last 
334     * match. For example:
335     * </p>
336     * <blockquote>
337     * <pre>
338     * MessageFormat mf = new MessageFormat("{0}, {0}, {0}");
339     * String forParsing = "x, y, z";
340     * Object[] objs = mf.parse(forParsing, new ParsePosition(0));
341     * // result now equals {new String("z")}
342     * </pre>
343     * </blockquote>
344     * <h4><a name="synchronization">Synchronization</a></h4>
345     * <p>
346     * Message formats are not synchronized. It is recommended to create separate
347     * format instances for each thread. If multiple threads access a format
348     * concurrently, it must be synchronized externally.
349     * </p>
350     * 
351     * @see java.util.Locale
352     * @see Format
353     * @see NumberFormat
354     * @see DecimalFormat
355     * @see ChoiceFormat
356     * @since Android 1.0
357     */

405        /**
406         * Changes this {@code MessageFormat} to use the specified pattern.
407         * 
408         * @param template
409         *            the new pattern.
410         * @exception IllegalArgumentException
411         *                if the pattern cannot be parsed.
412         * @since Android 1.0
413         */
414        public void applyPattern(String template) {
415            int length = template.length();
416            StringBuffer buffer = new StringBuffer();
417            ParsePosition position = new ParsePosition(0);
418            Vector<String> localStrings = new Vector<String>();
419            int argCount = 0;
420            int[] args = new int[10];
421            int maxArg = -1;
422            Vector<Format> localFormats = new Vector<Format>();
423            while (position.getIndex() < length) {
424                if (Format.upTo(template, position, buffer, '{')) {
425                    byte arg;
426                    int offset = position.getIndex();
427                    if (offset >= length
428                            || (arg = (byte) Character.digit(template
429                                    .charAt(offset++), 10)) == -1) {
430                        // text.19=Invalid argument number
431                        throw new IllegalArgumentException(Messages
432                                .getString("text.19")); //$NON-NLS-1$
433                    }
434                    position.setIndex(offset);
435                    localFormats.addElement(parseVariable(template, position));
436                    if (argCount >= args.length) {
437                        int[] newArgs = new int[args.length * 2];
438                        System.arraycopy(args, 0, newArgs, 0, args.length);
439                        args = newArgs;
440                    }
441                    args[argCount++] = arg;
442                    if (arg > maxArg) {
443                        maxArg = arg;
444                    }
445                }
446                localStrings.addElement(buffer.toString());
447                buffer.setLength(0);
448            }
449            this.strings = new String[localStrings.size()];
450            for (int i = 0; i < localStrings.size(); i++) {
451                this.strings[i] = localStrings.elementAt(i);
452            }
453            argumentNumbers = args;
454            this.formats = new Format[argCount];
455            for (int i = 0; i < argCount; i++) {
456                this.formats[i] = localFormats.elementAt(i);
457            }
458            maxOffset = argCount - 1;
459            maxArgumentIndex = maxArg;
460        }
461    
462        /**
463         * Returns a new instance of {@code MessageFormat} with the same pattern and
464         * formats as this {@code MessageFormat}.
465         * 
466         * @return a shallow copy of this {@code MessageFormat}.
467         * @see java.lang.Cloneable
468         * @since Android 1.0
469         */
470        @Override
471        public Object clone() {
472            MessageFormat clone = (MessageFormat) super.clone();
473            Format[] array = new Format[formats.length];
474            for (int i = formats.length; --i >= 0;) {
475                if (formats[i] != null) {
476                    array[i] = (Format) formats[i].clone();
477                }
478            }
479            clone.formats = array;
480            return clone;
481        }

483        /**
484         * Compares the specified object to this {@code MessageFormat} and indicates
485         * if they are equal. In order to be equal, {@code object} must be an
486         * instance of {@code MessageFormat} and have the same pattern.
487         * 
488         * @param object
489         *            the object to compare with this object.
490         * @return {@code true} if the specified object is equal to this
491         *         {@code MessageFormat}; {@code false} otherwise.
492         * @see #hashCode
493         * @since Android 1.0
494         */
495        @Override
496        public boolean equals(Object object) {
497            if (this == object) {
498                return true;
499            }
500            if (!(object instanceof MessageFormat)) {
501                return false;
502            }
503            MessageFormat format = (MessageFormat) object;
504            if (maxOffset != format.maxOffset) {
505                return false;
506            }
507            // Must use a loop since the lengths may be different due
508            // to serialization cross-loading
509            for (int i = 0; i <= maxOffset; i++) {
510                if (argumentNumbers[i] != format.argumentNumbers[i]) {
511                    return false;
512                }
513            }
514            return locale.equals(format.locale)
515                    && Arrays.equals(strings, format.strings)
516                    && Arrays.equals(formats, format.formats);
517        }
518    
519        /**
520         * Formats the specified object using the rules of this message format and
521         * returns an {@code AttributedCharacterIterator} with the formatted message and
522         * attributes. The {@code AttributedCharacterIterator} returned also includes the
523         * attributes from the formats of this message format.
524         * 
525         * @param object
526         *            the object to format.
527         * @return an {@code AttributedCharacterIterator} with the formatted message and
528         *         attributes.
529         * @exception IllegalArgumentException
530         *                if the arguments in the object array cannot be formatted
531         *                by this message format.
532         * @since Android 1.0
533         */
534        @Override
535        public AttributedCharacterIterator formatToCharacterIterator(Object object) {
536            if (object == null) {
537                throw new NullPointerException();
538            }
539    
540            StringBuffer buffer = new StringBuffer();
541            Vector<FieldContainer> fields = new Vector<FieldContainer>();
542    
543            // format the message, and find fields
544            formatImpl((Object[]) object, buffer, new FieldPosition(0), fields);
545    
546            // create an AttributedString with the formatted buffer
547            AttributedString as = new AttributedString(buffer.toString());
548    
549            // add MessageFormat field attributes and values to the AttributedString
550            for (int i = 0; i < fields.size(); i++) {
551                FieldContainer fc = fields.elementAt(i);
552                as.addAttribute(fc.attribute, fc.value, fc.start, fc.end);
553            }
554    
555            // return the CharacterIterator from AttributedString
556            return as.getIterator();
557        }

559        /**
560         * Converts the specified objects into a string which it appends to the
561         * specified string buffer using the pattern of this message format.
562         * <p>
563         * If the {@code field} member of the specified {@code FieldPosition} is
564         * {@code MessageFormat.Field.ARGUMENT}, then the begin and end index of
565         * this field position is set to the location of the first occurrence of a
566         * message format argument. Otherwise, the {@code FieldPosition} is ignored.
567         * </p>
568         * 
569         * @param objects
570         *            the array of objects to format.
571         * @param buffer
572         *            the target string buffer to append the formatted message to.
573         * @param field
574         *            on input: an optional alignment field; on output: the offsets
575         *            of the alignment field in the formatted text.
576         * @return the string buffer.
577         * @since Android 1.0
578         */
579        public final StringBuffer format(Object[] objects, StringBuffer buffer,
580                FieldPosition field) {
581            return formatImpl(objects, buffer, field, null);
582        }
583    
584        private StringBuffer formatImpl(Object[] objects, StringBuffer buffer,
585                FieldPosition position, Vector<FieldContainer> fields) {
586            FieldPosition passedField = new FieldPosition(0);
587            for (int i = 0; i <= maxOffset; i++) {
588                buffer.append(strings[i]);
589                int begin = buffer.length();
590                Object arg;
591                if (objects != null && argumentNumbers[i] < objects.length) {
592                    arg = objects[argumentNumbers[i]];
593                } else {
594                    buffer.append('{');
595                    buffer.append(argumentNumbers[i]);
596                    buffer.append('}');
597                    handleArgumentField(begin, buffer.length(), argumentNumbers[i],
598                            position, fields);
599                    continue;
600                }
601                Format format = formats[i];
602                if (format == null || arg == null) {
603                    if (arg instanceof Number) {
604                        format = NumberFormat.getInstance();
605                    } else if (arg instanceof Date) {
606                        format = DateFormat.getInstance();
607                    } else {
608                        buffer.append(arg);
609                        handleArgumentField(begin, buffer.length(),
610                                argumentNumbers[i], position, fields);
611                        continue;
612                    }
613                }
614                if (format instanceof ChoiceFormat) {
615                    String result = format.format(arg);
616                    MessageFormat mf = new MessageFormat(result);
617                    mf.setLocale(locale);
618                    mf.format(objects, buffer, passedField);
619                    handleArgumentField(begin, buffer.length(), argumentNumbers[i],
620                            position, fields);
621                    handleformat(format, arg, begin, fields);
622                } else {
623                    format.format(arg, buffer, passedField);
624                    handleArgumentField(begin, buffer.length(), argumentNumbers[i],
625                            position, fields);
626                    handleformat(format, arg, begin, fields);
627                }
628            }
629            if (maxOffset + 1 < strings.length) {
630                buffer.append(strings[maxOffset + 1]);
631            }
632            return buffer;
633        }

636         * Adds a new FieldContainer with MessageFormat.Field.ARGUMENT field,

682        /**
683         * If fields vector is not null, find and add the fields of this format to
684         * the fields vector by iterating through its AttributedCharacterIterator
685         * 
686         * @param format
687         *            the format to find fields for
688         * @param arg
689         *            object to format
690         * @param begin
691         *            the index where the string this format has formatted begins
692         * @param fields
693         *            fields vector, each entry in this vector are of type
694         *            FieldContainer.
695         */
696        private void handleformat(Format format, Object arg, int begin,
697                Vector<FieldContainer> fields) {
698            if (fields != null) {
699                AttributedCharacterIterator iterator = format
700                        .formatToCharacterIterator(arg);
701                while (iterator.getIndex() != iterator.getEndIndex()) {
702                    int start = iterator.getRunStart();
703                    int end = iterator.getRunLimit();
704    
705                    Iterator<?> it = iterator.getAttributes().keySet().iterator();
706                    while (it.hasNext()) {
707                        AttributedCharacterIterator.Attribute attribute = (AttributedCharacterIterator.Attribute) it
708                                .next();
709                        Object value = iterator.getAttribute(attribute);
710                        fields.add(new FieldContainer(begin + start, begin + end,
711                                attribute, value));
712                    }
713                    iterator.setIndex(end);
714                }
715            }
716        }

885        /**
886         * Parses the message argument from the specified string starting at the
887         * index specified by {@code position}. If the string is successfully
888         * parsed then the index of the {@code ParsePosition} is updated to the
889         * index following the parsed text. On error, the index is unchanged and the
890         * error index of {@code ParsePosition} is set to the index where the error
891         * occurred.
892         * 
893         * @param string
894         *            the string to parse.
895         * @param position
896         *            input/output parameter, specifies the start index in
897         *            {@code string} from where to start parsing. If parsing is
898         *            successful, it is updated with the index following the parsed
899         *            text; on error, the index is unchanged and the error index is
900         *            set to the index where the error occurred.
901         * @return the array of objects resulting from the parse, or {@code null} if
902         *         there is an error.
903         * @since Android 1.0
904         */
905        public Object[] parse(String string, ParsePosition position) {
906            if (string == null) {
907                return new Object[0];
908            }
909            ParsePosition internalPos = new ParsePosition(0);
910            int offset = position.getIndex();
911            Object[] result = new Object[maxArgumentIndex + 1];
912            for (int i = 0; i <= maxOffset; i++) {
913                String sub = strings[i];
914                if (!string.startsWith(sub, offset)) {
915                    position.setErrorIndex(offset);
916                    return null;
917                }
918                offset += sub.length();
919                Object parse;
920                Format format = formats[i];
921                if (format == null) {
922                    if (i + 1 < strings.length) {
923                        int next = string.indexOf(strings[i + 1], offset);
924                        if (next == -1) {
925                            position.setErrorIndex(offset);
926                            return null;
927                        }
928                        parse = string.substring(offset, next);
929                        offset = next;
930                    } else {
931                        parse = string.substring(offset);
932                        offset = string.length();
933                    }
934                } else {
935                    internalPos.setIndex(offset);
936                    parse = format.parseObject(string, internalPos);
937                    if (internalPos.getErrorIndex() != -1) {
938                        position.setErrorIndex(offset);
939                        return null;
940                    }
941                    offset = internalPos.getIndex();
942                }
943                result[argumentNumbers[i]] = parse;
944            }
945            if (maxOffset + 1 < strings.length) {
946                String sub = strings[maxOffset + 1];
947                if (!string.startsWith(sub, offset)) {
948                    position.setErrorIndex(offset);
949                    return null;
950                }
951                offset += sub.length();
952            }
953            position.setIndex(offset);
954            return result;
955        }

982        private int match(String string, ParsePosition position, boolean last,
983                String[] tokens) {
984            int length = string.length(), offset = position.getIndex(), token = -1;
985            while (offset < length && Character.isWhitespace(string.charAt(offset))) {
986                offset++;
987            }
988            for (int i = tokens.length; --i >= 0;) {
989                if (string.regionMatches(true, offset, tokens[i], 0, tokens[i]
990                        .length())) {
991                    token = i;
992                    break;
993                }
994            }
995            if (token == -1) {
996                return -1;
997            }
998            offset += tokens[token].length();
999            while (offset < length && Character.isWhitespace(string.charAt(offset))) {
1000                offset++;
1001            }
1002            char ch;
1003            if (offset < length
1004                    && ((ch = string.charAt(offset)) == '}' || (!last && ch == ','))) {
1005                position.setIndex(offset + 1);
1006                return token;
1007            }
1008            return -1;
1009        }

1011        private Format parseVariable(String string, ParsePosition position) {
1012            int length = string.length(), offset = position.getIndex();
1013            char ch;
1014            if (offset >= length
1015                    || ((ch = string.charAt(offset++)) != '}' && ch != ',')) {
1016                // text.15=Missing element format
1017                throw new IllegalArgumentException(Messages.getString("text.15")); //$NON-NLS-1$
1018            }
1019            position.setIndex(offset);
1020            if (ch == '}') {
1021                return null;
1022            }
1023            int type = match(string, position, false, new String[] { "time", //$NON-NLS-1$
1024                    "date", "number", "choice" }); //$NON-NLS-1$ //$NON-NLS-2$ //$NON-NLS-3$
1025            if (type == -1) {
1026                // text.16=Unknown element format
1027                throw new IllegalArgumentException(Messages.getString("text.16")); //$NON-NLS-1$
1028            }
1029            StringBuffer buffer = new StringBuffer();
1030            ch = string.charAt(position.getIndex() - 1);
1031            switch (type) {
1032                case 0: // time
1033                case 1: // date
1034                    if (ch == '}') {
1035                        return type == 1 ? DateFormat.getDateInstance(
1036                                DateFormat.DEFAULT, locale) : DateFormat
1037                                .getTimeInstance(DateFormat.DEFAULT, locale);
1038                    }
1039                    int dateStyle = match(string, position, true, new String[] {
1040                            "full", "long", "medium", "short" }); //$NON-NLS-1$ //$NON-NLS-2$ //$NON-NLS-3$ //$NON-NLS-4$
1041                    if (dateStyle == -1) {
1042                        Format.upToWithQuotes(string, position, buffer, '}', '{');
1043                        return new SimpleDateFormat(buffer.toString(), locale);
1044                    }
1045                    switch (dateStyle) {
1046                        case 0:
1047                            dateStyle = DateFormat.FULL;
1048                            break;
1049                        case 1:
1050                            dateStyle = DateFormat.LONG;
1051                            break;
1052                        case 2:
1053                            dateStyle = DateFormat.MEDIUM;
1054                            break;
1055                        case 3:
1056                            dateStyle = DateFormat.SHORT;
1057                            break;
1058                    }
1059                    return type == 1 ? DateFormat
1060                            .getDateInstance(dateStyle, locale) : DateFormat
1061                            .getTimeInstance(dateStyle, locale);
1062                case 2: // number
1063                    if (ch == '}') {
1064                        // BEGIN android-changed
1065                        return NumberFormat.getInstance(locale);
1066                        // END android-changed
1067                    }
1068                    int numberStyle = match(string, position, true, new String[] {
1069                            "currency", "percent", "integer" }); //$NON-NLS-1$ //$NON-NLS-2$ //$NON-NLS-3$
1070                    if (numberStyle == -1) {
1071                        Format.upToWithQuotes(string, position, buffer, '}', '{');
1072                        return new DecimalFormat(buffer.toString(),
1073                                new DecimalFormatSymbols(locale));
1074                    }
1075                    switch (numberStyle) {
1076                        case 0: // currency
1077                            return NumberFormat.getCurrencyInstance(locale);
1078                        case 1: // percent
1079                            return NumberFormat.getPercentInstance(locale);
1080                    }
1081                    return NumberFormat.getIntegerInstance(locale);
1082            }
1083            // choice
1084            try {
1085                Format.upToWithQuotes(string, position, buffer, '}', '{');
1086            } catch (IllegalArgumentException e) {
1087                // ignored
1088            }
1089            return new ChoiceFormat(buffer.toString());
1090        }

1215        /**
1216         * Returns the pattern of this message format.
1217         * 
1218         * @return the pattern.
1219         * @since Android 1.0
1220         */
1221        public String toPattern() {
1222            StringBuffer buffer = new StringBuffer();
1223            for (int i = 0; i <= maxOffset; i++) {
1224                appendQuoted(buffer, strings[i]);
1225                buffer.append('{');
1226                buffer.append(argumentNumbers[i]);
1227                Format format = formats[i];
1228                String pattern = null;
1229                if (format instanceof ChoiceFormat) {
1230                    buffer.append(",choice,"); //$NON-NLS-1$
1231                    pattern = ((ChoiceFormat) format).toPattern();
1232                } else if (format instanceof DecimalFormat) {
1233                    pattern = decodeDecimalFormat(buffer, format);
1234                } else if (format instanceof SimpleDateFormat) {
1235                    pattern = decodeSimpleDateFormat(buffer, format);
1236                } else if (format != null) {
1237                    // text.17=Unknown format
1238                    throw new IllegalArgumentException(Messages
1239                            .getString("text.17")); //$NON-NLS-1$
1240                }
1241                if (pattern != null) {
1242                    boolean quote = false;
1243                    int index = 0, length = pattern.length(), count = 0;
1244                    while (index < length) {
1245                        char ch = pattern.charAt(index++);
1246                        if (ch == '\'') {
1247                            quote = !quote;
1248                        }
1249                        if (!quote) {
1250                            if (ch == '{') {
1251                                count++;
1252                            }
1253                            if (ch == '}') {
1254                                if (count > 0) {
1255                                    count--;
1256                                } else {
1257                                    buffer.append("'}"); //$NON-NLS-1$
1258                                    ch = '\'';
1259                                }
1260                            }
1261                        }
1262                        buffer.append(ch);
1263                    }
1264                }
1265                buffer.append('}');
1266            }
1267            if (maxOffset + 1 < strings.length) {
1268                appendQuoted(buffer, strings[maxOffset + 1]);
1269            }
1270            return buffer.toString();
1271        }

1273        private void appendQuoted(StringBuffer buffer, String string) {
1274            int length = string.length();
1275            for (int i = 0; i < length; i++) {
1276                char ch = string.charAt(i);
1277                if (ch == '{' || ch == '}') {
1278                    buffer.append('\'');
1279                    buffer.append(ch);
1280                    buffer.append('\'');
1281                } else {
1282                    buffer.append(ch);
1283                }
1284            }
1285        }
